# Padrões de Bugs Recorrentes no Antigravity

Este arquivo cataloga bugs já vistos no projeto e suas abordagens de resolução.
Atualize sempre que um bug recorrente for identificado e resolvido.

---

## Como usar este arquivo

Consulte quando:
- O bug relatado parece familiar ou já foi visto antes
- O usuário relata que "esse bug voltou"
- Você identificou um padrão sintomático conhecido

---

## Catálogo de Bugs

> ⚠️ Este arquivo começa vazio e deve ser populado conforme bugs são identificados e resolvidos no projeto Antigravity.

### Template para adicionar novo bug

```
### [Nome descritivo do bug]
**Sintoma:** O que o usuário vê/reporta
**Causa raiz:** Por que acontece
**Arquivo(s) comumente afetados:** caminho/do/arquivo
**Correção padrão:** Descrição objetiva da solução
**Primeira ocorrência:** [data ou versão]
**Recorrências:** [quantas vezes reapareceu e por quê]
```
